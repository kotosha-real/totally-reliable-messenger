## Description

We strive to create a totally reliable messenger for all the comrades. Have not succeed yet though. But we will.

## Jumpstart

First of all, install dependencies with
```javascript
npm install
```

Start your work with
```javascript
npm start
```

Build production-ready version with
```javascript
npm run build
```

[Netlify](https://quizzical-wiles-72addf.netlify.app/) autodeploy configured from `deploy` branch
